# Temperature converter
Temperature converter app which converts Celsius to Kelvin, Farenheit, Romer and vice versa with .NET Core API 3.1
Supported conversions:
C->K (param "ck")
C->F (param "cf)
K->C (param "kc)
K->F (param "kf)
F->C (param "fc")
R->C (param "rc")
C->R (param "cr")
R->F (param "rf")
F->R (param "Fr)
