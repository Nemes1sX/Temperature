# Temperature converter
Temperature converter app which converts Celsius to Kelvin, Farenheit, Romer and vice versa with .NET Core API 3.1 <br/>
Supported conversions: <br/>
C->K (param "ck") <br/>
C->F (param "cf) <br/>
K->C (param "kc) <br/>
K->F (param "kf) <br/>
F->C (param "fc") <br/>
R->C (param "rc") <br/>
C->R (param "cr") <br/>
R->F (param "rf") <br/>
F->R (param "Fr) <br/>
